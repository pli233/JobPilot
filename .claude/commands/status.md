# /status - 查看申请状态

查看求职进度和统计的快捷命令。

## 使用方式

```
/status          # 显示总体统计
/status today    # 今日申请
/status week     # 本周统计
/status pending  # 待处理申请
```

## 执行流程

1. 查询 SQLite 数据库 (`data/db/jobpilot.db`)
2. 计算各状态申请数量
3. 生成统计报告

## 输出示例

```
📊 求职进度统计

总申请数: 45
├── 待回复: 30 (67%)
├── 面试中: 8 (18%)
├── 已拒绝: 5 (11%)
└── 已录用: 2 (4%)

📅 本周活动
├── 新申请: 12
├── 收到回复: 5
└── 安排面试: 3

🎯 匹配度分析
├── 90%+ 匹配: 15 个职位
├── 70-90% 匹配: 20 个职位
└── <70% 匹配: 10 个职位

💡 建议
- 有 5 个高匹配度职位尚未申请
- 上次申请是 2 天前，建议保持节奏
```

## 相关命令

- `/search` - 搜索新职位
- `/daily` - 执行每日例行任务
