# /daily - 每日例行任务

执行每日求职例行任务的快捷命令。

## 使用方式

```
/daily           # 执行完整每日任务
/daily search    # 只执行搜索
/daily review    # 只检查状态更新
```

## 执行流程

### 1. 搜索新职位
- 读取 `config/preferences.json` 中的搜索关键词
- 并行搜索三大平台
- 与已保存职位去重
- 计算匹配度并保存高匹配职位

### 2. 检查状态更新
- 查询 "待回复" 状态的申请
- 检查是否有新邮件/消息
- 更新申请状态

### 3. 生成日报
```
📋 每日求职报告 - 2026-01-28

🔍 搜索结果
├── 新发现职位: 23 个
├── 高匹配度 (90%+): 5 个
└── 已保存: 8 个

📬 状态更新
├── 新回复: 2 个
├── 面试邀请: 1 个
└── 拒信: 1 个

📊 总体进度
├── 本周申请: 15 个
├── 面试中: 3 个
└── 等待回复: 25 个

✅ 建议行动
1. 申请 5 个新发现的高匹配职位
2. 跟进 ABC 公司面试安排
3. 更新 resume_main.pdf 添加最新项目
```

## 定时执行

可配合系统定时任务每日自动执行：
- Windows: Task Scheduler
- Mac/Linux: cron

## 相关命令

- `/search` - 手动搜索
- `/status` - 查看详细状态
